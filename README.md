Promts usados:

1\) Necesito que crees un Gestor de Tareas(TaskFlow). Un formulario para agregar tareas (título, prioridad, fecha límite). Validación de campos vacíos y formato de fecha. Los datos se almacenan en un arreglo de objetos y se renderizan en una lista dinámica. Que debe estar relacionado con el instituto Inacap. 2\) Puedes agregar un historial de las tareas eliminadas, con tambien tener la capacidad de eliminar definitivamente las tareas de ese historial.
